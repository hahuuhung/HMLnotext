# Tài liệu nghiệm thu dự án (Walkthrough) - Phase 1: Prototype

Tài liệu này tổng hợp kết quả triển khai Phase 1 cho ứng dụng **No-code Video Workflow Builder** (Giao diện thiết kế video tự động dạng kéo thả tương tự n8n).

## Các thay đổi đã thực hiện

1. **Khởi tạo và Cấu trúc dự án**:
   - Dự án được xây dựng trên nền tảng **Vite + React (TypeScript)**.
   - Thiết lập các thư mục tài liệu thiết kế và sơ đồ quy trình tại [doc/](file:///d:/AntiGravity/HML/doc).

2. **Giao diện & Thiết kế hệ thống (UI/UX)**:
   - Toàn bộ giao diện hiển thị bằng **Tiếng Việt** chuẩn hóa.
   - Bố cục gồm 4 phân khu chính:
     - **Thanh thư viện Node bên trái**: Chứa các node Trigger, Input, AI, Visual, Render hỗ trợ kéo thả.
     - **Màn hình Canvas React Flow ở giữa**: Nơi kết nối luồng chạy.
     - **Bảng thuộc tính bên phải (Inspector)**: Hiển thị form thay đổi cấu hình cho từng node được chọn.
     - **Bảng điều khiển dưới cùng**: Hiển thị Timeline phân cảnh video trực quan, màn hình trình phát Preview sinh động và bảng Console in log.

3. **Mô phỏng chạy Workflow & Render**:
   - Khi bấm nút **Chạy thử (Run)**, hệ thống mô phỏng việc chạy tuần tự từ Trigger -> Đọc chủ đề -> Phân tích kịch bản bằng AI -> Vẽ hình ảnh -> Dựng video.
   - Các node sáng lên tương ứng với trạng thái (Đang chờ, Đang chạy, Thành công).
   - Tải về cấu hình manifest kịch bản video qua nút **Tải video (MP4)** dưới định dạng tệp tin.
